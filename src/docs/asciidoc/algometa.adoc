== 1. 개요

알고리즘 공부에 있어서 메타인지(메타 인지: 본인이 알고 있다고 생각하는 내용을 정확히 인지하는 능력)
는 매우 중요합니다. 그 이유는 알고 있다고 생각해도 정확히 이해하고 응용하지 않으면 문제를 결코 풀 수
없기 때문입니다. 그렇기 때문에 복습과 정확한 이해가 중요하지만 대부분 사람들의 경우 알고리즘을 한번만 풀고
복습하지 않습니다. 그래서 실제 알고 있다고 생각하는 내용과 테스트 볼 때 잘 풀리지 않는 경우가 있을 수 있습니다.

Algo meta는 알고리즘 푼 문제에 대해서 기록하고 개인이 올려놓은 문제 정보를 바탕으로 개인 학습 맞춤
서비스를 제공합니다. 해당 API에서 제공하는 것은 크게 3가지 입니다.

1. 개발자에게 좀 더 친숙한 블로그 형식으로 알고리즘 문제를 기록하고 관리
2. 기록한 문제를 다시 복습하면 히스토리에 코드 정보가 저장됨(과거의 풀이 이력을 알 수 있음)
3. 개인 맞춤 서비스 제공(오랫동안 풀지 않았거나 어려웠다고 기록한 문제 복습 가능하도록 알림,
또는 비슷한 유형의 문제 추천)
